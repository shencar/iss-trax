## Running the app

To run the app, you need to have Python3.7+ installed. Then, install the dependencies:

```bash
pip install -r requirements.txt
```

Then, run the app:

```bash
cd backend
python server.py
```

Now, open your browser at http://localhost:4000

Enjoy!
